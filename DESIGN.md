## Brand & Style

The design system is anchored in the concept of "Intelligent Calm." It balances the rigorous, structured nature of language learning with the fluid, adaptive power of AI. Drawing inspiration from industry leaders like Linear and Stripe, the aesthetic is deeply rooted in **Modern SaaS Minimalism**. 

The UI should feel like a premium tool for cognitive growth—uncluttered, fast, and precise. It avoids the "gamified" clutter common in education apps, opting instead for a professional atmosphere that respects the learner's focus. Visual interest is generated through sophisticated typography, expansive whitespace, and ethereal ambient glows that signify AI activity. The target audience is the modern professional or dedicated student who values efficiency and a high-tier digital experience.

## Colors

The palette is built around four layers: foundation, brand, AI accents, and semantic/gamification.

- **Primary Blue (#1B75BC):** Derived from the logo brackets. Used for core actions, progress indicators, and brand-critical elements. Hover state: `#1565a8`. Active/pressed: `#005c99`.
- **AI Accents (Cyan & Purple):** Reserved exclusively for AI-driven features — smart suggestions, speech analysis, personalized pathways. Appear as subtle gradients or soft outer glows rather than solid blocks.
- **Neutrals:** Cool-toned off-white (`#f9fafb`) base aligned with the logo's neutral feel. Deep anthracite (`#1e1b1c`) for headlines, softer slate (`#414751`) for body copy.
- **Feedback Colors:** Distinct from brand colors. Green for correct answers, a brighter red (`#dc2626`) for wrong-answer banners (separate from the form-error `#ba1a1a`), amber for time warnings, purple for AI hints.
- **Activity Colors:** Each exercise type has a dedicated color pair (solid + light background) for card headers, progress rings, and category badges.
- **Gamification Colors:** Hearts, streaks, coins, combo multipliers, and league tiers each have a fixed color to build visual muscle memory.

## Typography

This design system utilizes **Inter** for all roles to maintain a systematic, utilitarian, and modern feel. 

Headlines utilize a bold weight with slightly tightened letter-spacing to create a strong visual "anchor" on the page. Body text is prioritized for legibility with a generous line-height (1.5x) to facilitate comfortable reading of foreign language characters. Labels use a medium to semi-bold weight and occasional uppercase styling to differentiate them from interactive content.

## Layout & Spacing

The layout is built on a **12-column fluid grid** with a fixed maximum width for desktop to prevent line lengths from becoming unreadable. 

The spacing rhythm follows an 8px base unit. 
- **Desktop:** Use 40px margins with 24px gutters. Elements should align to the grid, but AI components can occasionally break the grid with "soft" edges or ambient background blurs.
- **Mobile:** Transition to a single-column layout with 16px side margins. 
- **Vertical Rhythm:** Generous vertical padding (64px to 120px) should be used between major sections to maintain the "Minimalist" brand promise and allow the UI to breathe.

## Elevation & Depth

Depth in the design system is achieved through **Glassmorphism** and **Ambient Shadows**. 

1.  **Surfaces:** Primary cards use a white background at 80-90% opacity with a `backdrop-filter: blur(12px)`. Borders are 1px thick, utilizing a soft slate-blue (#E2E8F0) to define edges without adding visual weight.
2.  **Shadows:** Use a "Linear-style" shadow stack: a very subtle, sharp inner border shadow combined with a soft, highly diffused outer shadow (Color: `rgba(0,0,0,0.04)`, Blur: `20px`).
3.  **AI Glows:** For AI-powered elements, use a "Secondary Layer" depth technique where a faint radial gradient of Cyan or Purple is placed *behind* the glass surface, creating a subtle luminescent effect that feels embedded rather than floating.

## Shapes

The design system uses a **Rounded** (Level 2) shape language. 

Standard components like buttons and input fields use a **0.5rem (8px)** corner radius. Larger containers and glassmorphic cards use **1rem (16px)** to create a nested, organic feel. This balance of roundedness ensures the platform feels approachable and modern without losing the professional edge required for a language learning tool.

## Components

### Buttons
- **Primary:** `#1b75bc` background, white text. Hover: `#1565a8`. No gradients, just a subtle 1px top-highlight for a tactile feel.
- **Ghost:** Transparent background with `#1b75bc` text. On hover, `#eff6ff` background appears.
- **AI Action:** Gradient border (Cyan → Purple) with glassmorphic background. Implement via `::before` pseudo-element, not `border-image`.
- **Destructive:** `#dc2626` background, white text. For delete/irreversible actions.
- **Success:** `#16a34a` background, white text. For completion confirmations.
- **Warning:** `#d97706` background, white text. For heart-spend confirmations, cautionary actions.
- **Disabled:** `#e5e7eb` background, `#9ca3af` text.

### Cards
- Always glassmorphic. 1px border (#E2E8F0). No solid fills for top-level containers. 

### Input Fields
- `#f9fafb` background with a 1px `#e5e7eb` border. On focus, border transitions to `#1b75bc` with a soft 4px glow (`rgba(27,117,188,0.15)`).

### Chips & Progress
- Progress bars: 4px height, `#1b75bc` fill. AI-verified progress: `#57dffe` (Cyan) fill.
- Chips: `#eef0f3` background, `#414751` text, `0.25rem` radius.

### Exercise Feedback Banners
- **Correct:** `#dcfce7` background, `#16a34a` border-left (4px), `#16a34a` icon.
- **Wrong:** `#fee2e2` background, `#dc2626` border-left (4px), `#dc2626` icon.
- **Hint (AI):** `#f5f3ff` background, `#7c3aed` border-left (4px).
- **Timeout warning:** `#fef3c7` background, `#d97706` border-left (4px).

### Special AI Components
- **The "Pulse":** For voice recording or AI thinking states, use a soft, breathing Cyan (`#57dffe`) glow behind the primary action button.
- **Smart Hints:** Small cards with `#7c3aed` left-border and `#f5f3ff` background tint to signify machine-generated assistance.
